# Uninstall Virtue FX Manager

Official uninstall instructions are blocked until the final package layout is confirmed.

The official release should document:

- installed REAPER extension file location
- user data location
- thumbnail/cache location
- how to preserve or remove favorites, notes, settings, themes, and thumbnails
- how to deactivate this computer before uninstalling

Deactivation should happen inside VFxM before removing the app when possible.
