# Virtue FX Manager Release Notes

## Official Release Preparation

Release artifacts are not included in this website repository yet.

When the first official VFxM build is available, publish:

- version
- channel: `stable`, `rc`, or `beta`
- platform
- file name
- SHA-256 checksum
- release date
- minimum supported REAPER version, if known
- installation notes
- fixed issues and known limitations

Do not mark a build as official until checkout, installer delivery, app-side activation, deactivation, validation, and offline grace have passed staging tests.
