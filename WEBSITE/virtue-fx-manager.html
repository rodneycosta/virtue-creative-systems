Features

Turn thousands of plugins into a fast visual workflow.

Virtue FX Manager combines plugin browser, favorites, thumbnails, notes, filters, categories, list/unlist control, and REAPER insertion in one focused tool.

Native REAPER integration

Native C++ extension opened through the action “Virtue: Show FX Manager,” with floating tool-window behavior inside REAPER.

Visual plugin browser

Searchable plugin table with listed, unlisted, and all views, status indicators, sorting, keyboard navigation, and multi-selection.

Developer, type, and category filters

Multi-column chip filters help narrow large libraries by developer, format family, instrument/effect role, and workflow category.

Favorites system

Multiple Favorites tabs, grid/list views, custom tab colors, thumbnail sizing, selection tools, moving, copying, and batch actions.

Thumbnail Manager

Large plugin preview, local thumbnail storage, image load/copy/paste, undo/redo, placeholders, and refresh across Favorites.

Notes per plugin

Keep multiline notes that follow the selected plugin, so you can remember why a tool matters and how you like to use it.

Direct track insertion

Insert plugins into selected tracks, create suitable tracks when needed, and support double-click, drag/drop, and batch insert workflows.

Safe list/unlist management

List or unlist plugins individually or in batches with REAPER cache backup before rewriting plugin cache files.

Themes and customization

Built-in themes, user themes, live color editing, font size control, thumbnail folders, sorting, and readable contrast correction.

Localization and personal data

English, Portuguese, Spanish, and fallback coverage where available, with Favorites, notes, settings, themes, and thumbnails kept local.

Compatibility

Built specifically for REAPER plugin libraries.

Virtue FX Manager is designed around REAPER’s plugin cache and resource-path workflow. It supports the plugin formats REAPER stores in its library, including VST, VST3, AU, CLAP, JSFX, and architecture/cache variants where REAPER stores them. Exact platform support, installation steps, and release builds will be published with the first public release.

FAQ

Questions before release.

What is Virtue FX Manager?

A native plugin manager for REAPER that helps users browse, organize, preview, favorite, thumbnail, note, filter, list/unlist, and insert plugins faster.

Is it a plugin?

It is intended as a native REAPER extension, not a browser-based plugin database. Final installer and delivery details will be published with the first public build.

Is it only for REAPER?

Yes. The first product is designed specifically around REAPER workflows, plugin cache data, resource paths, and track insertion behavior.
