System requirements

REAPER is required.

VFxM runs as a native C++ extension inside Cockos REAPER. Check compatibility below before installing.

  • REAPER 6.0 or newer is required.
  • macOS: macOS 11 (Big Sur) or newer, native Apple Silicon & Intel.
  • Windows: Windows 10 & 11, standard 64-bit.
