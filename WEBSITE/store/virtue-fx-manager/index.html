Visual Plugin Manager for REAPER

A focused REAPER-native workflow accelerator for large plugin libraries.

Virtue FX Manager is a professional visual plugin manager for REAPER. Browse your entire plugin collection, search by name, developer, type, or category, organize favorites into custom tabs, rate plugins, attach notes and thumbnails, hide clutter, and insert plugins or saved FX chains by double-clicking or dragging directly into REAPER.

  • Visual plugin browser Thumbnails and fast recognition
  • Search and filters Developer, type, category, and status
  • Favorites tabs Custom boards for real studio workflows
  • Ratings and notes Personal recall for your best tools
  • FX presets and chains Reusable REAPER processing setups
  • List/unlist visibility Non-destructive clutter control
