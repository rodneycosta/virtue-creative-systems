Activate inside VFxM.

The website handles purchase, downloads, documentation, and support. License entry, activation, validation, offline grace, and deactivation happen inside Virtue FX Manager in REAPER. Activation happens inside the app, not on a public website page.

No website activation form

Activation flow

  1. Buy Virtue FX Manager through the store checkout when it is live.
  2. Receive the license/passkey from the store provider by email or provider account page.
  3. Download and install VFxM.
  4. Open REAPER and launch VFxM.
  5. Enter the license/passkey inside the VFxM activation dialog.
  6. VFxM contacts the secure license backend and stores a signed local entitlement if activation succeeds.
Offline

Grace period

After successful online validation, VFxM can continue during the configured offline grace window.

Move

Deactivate in-app

Device deactivation is an app feature. VFxM will deactivate only this computer when the official app flow is implemented.

Support

Diagnostics

VFxM diagnostics should include status and timestamps, never raw license keys or raw machine identifiers.
