This page will hold final installation steps for the official VFxM release package. Platform details will be published only after tested release artifacts exist.

Expected install flow

  1. Download the official VFxM package from the download page.
  2. Verify the SHA-256 checksum if one is published.
  3. Run the installer or follow the platform-specific manual install steps.
  4. Open REAPER and launch Virtue FX Manager from the installed REAPER action.
  5. Enter your license/passkey inside VFxM when prompted.
