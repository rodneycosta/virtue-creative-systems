FAQ

Questions before purchase.

Is Virtue FX Manager a plugin?

Virtue FX Manager is a REAPER-native plugin-management workflow tool/extension. It is designed to help you browse, organize, favorite, rate, thumbnail, note, and insert plugins inside REAPER workflows.

Does it work with other DAWs?

The current product is designed for REAPER. Do not buy or test it expecting Ableton, Logic, Cubase, Studio One, Bitwig, or Pro Tools support unless a future version explicitly says so.

Does VFxM delete my plugins?

No. Visibility features are designed to hide, list, or unlist plugins without deleting plugin files from disk.

Can I save plugin presets?

VFxM can save favorite FX preset-style setups and FX chains from REAPER, then restore them as reusable visual favorites.

Can I organize plugins by category?

Yes. VFxM supports developer, type, and category filters, plus category color cues and optional list colorization.

Can I use thumbnails?

Yes. VFxM supports plugin thumbnails, large previews, favorite thumbnails, custom images, clipboard paste, and capture workflows where available.

Is it finished?
